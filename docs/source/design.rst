Design
======

API names are made to look similar to current pandas APIs.

Grouping operations

- renaming
- dropping
- pipe

Our lambdas API:

- converting types
- mapping
- binning for numerical values
- binning for categorical values
- applying
- sapplying
- dapplying
- dropping columns with rules

Extras:

- series
    - scaler
    - pctg
    - binning for numerical values
    - binning for categorical values
    - ascii plotting
